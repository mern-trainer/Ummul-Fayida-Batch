commands

-> git --version
-> git init - initialize local git repo.

-> git config --global user.name "your name"
-> git config --global user.email "your email"

-> git add . -> add all files
-> git add <file_1> <file_2> - add specific files
-> git commit -m "change message"

-> git remote add <remote_name> <remote_url> -> set new remote
-> git config --get remote.<remote_name>.url -> get remote url

-> git push <remote_name> <branch_name> -> push changes to github
